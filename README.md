# 뮤지컬 할인소식을 알려주는 나만의 게시판
Spring Boot를 사용해 개발한 간단한 게시판 서비스입니다.
<br/>
<br/>

## 📑 프로젝트 소개
계명대학교의 최신 뮤지컬 할인 정보를 알려주는 게시판 서비스
<br/> 
<br/>


## 🤔 프로젝트 배경
매번 학교 홈페이지를 들어가서 최신 뮤지컬 할인 정보를 검색하기가 너무 번거로워 만들었습니다.
